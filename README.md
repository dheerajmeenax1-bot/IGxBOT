# 🚀 Instagram Mass Message Bot v3.0

Ek powerful Telegram bot jo Instagram par mass messages bhejta hai with parallel processing (50 workers).

## ✨ Features
- **⚡ Multiple Speeds:** Ultra (3000/min), Turbo, Fast, aur Normal options.
- **🔄 Parallel Sending:** Ek saath 50 workers ka istemal message bhejne ke liye.
- **🍪 Session Login:** Browser ki `sessionid` use karke safe login.
- **🔐 Private Access:** Sirf Authorized users hi bot use kar sakte hain.

## 🛠️ Setup Instructions

1. **GitHub Secrets Set Karein:**
   - Go to `Settings` > `Secrets and variables` > `Actions`.
   - Click `New repository secret`.
   - Name: `BOT_TOKEN`
   - Value: (Apna Telegram Bot Token yahan dalein).

2. **Bot Start Karein:**
   - Go to `Actions` tab.
   - Select `Run Instagram Bot`.
   - Click `Run workflow`.

3. **Telegram Par Use Karein:**
   - Send `/start` to your bot.
   - Use `Session ID Login` button.
   - Set target, message, and speed.
   - Hit **START**!
